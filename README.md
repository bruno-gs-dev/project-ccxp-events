# Project-CCXP-Events

## Descrição

Este projeto é uma landing page para a CCXP, um dos maiores eventos de cultura pop do mundo. A página foi desenvolvida para ser informativa, atraente e fácil de usar, fornecendo aos visitantes todas as informações essenciais sobre o evento.

## Funcionalidades

- **Contagem Regressiva:** Um cronômetro que mostra quanto tempo falta para o início do evento.
- **Vídeo de Destaque:** Um vídeo que captura a energia e a emoção da CCXP.
- **Ingressos:** Seções detalhadas com os diferentes tipos de ingressos, seus preços e benefícios.
- **Galeria de Imagens:** Uma coleção de imagens de eventos anteriores para dar aos visitantes um gostinho do que esperar.

## Tecnologias Utilizadas

- **HTML5:** Para a estrutura da página.
- **CSS3:** Para a estilização e o design.
- **JavaScript:** Para a funcionalidade interativa, como a contagem regressiva.

## Como Executar o Projeto

1. Clone o repositório:
   ```bash
   git clone https://github.com/seu-usuario/project-ccxp-events.git
   ```
2. Navegue até o diretório do projeto:
   ```bash
   cd project-ccxp-events
   ```
3. Abra o arquivo `index.html` no seu navegador de preferência.
